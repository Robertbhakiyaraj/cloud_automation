###  Access UI with web.xml testMode=true
Install Google Chrome header extension

Set the following headers:

* rolelist: Wire Department Administrator,Payments Operator Level 3,ALL,AZ MASTER ROLE
* WL-Proxy-SSL: true
* username: Manager_US
